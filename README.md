# 学生作业相似度评分系统

基于语义向量分析和图片相似度计算的学生作业评分工具，用于检测学生答案的原创性并按相似度进行评分。

## 功能特性

- **语义相似度分析**：使用 jieba 分词和 TF-IDF 算法提取文本特征向量
- **图片相似度检测**：支持从 Word 文档中提取图片并计算相似度
- **综合评分机制**：文本相似度（80%）+ 图片相似度（20%）
- **正态分布评分**：按照正态分布比例划分分数区间
- **Excel 成绩回填**：自动更新学生成绩到 Excel 文件
- **智能过滤**：自动删除状态为"未交"的学生记录

```bash
python score_class.py <文件夹路径>
```

### 示例

```bash
python score_class.py 23-3
```

## 输入文件要求

### 学生作业文件
- 格式：`.doc` 或 `.docx`
- 命名规范：`<学院>-<专业>-<班级>-<学号>-<姓名>.doc`
- 内容要求：包含"学生答案："与"正确答案："中间标记的答案部分

### Excel 成绩文件
- 格式：`.xls` 或 `.xlsx`
- 必需列：学号、分数（或成绩/得分）
- 可选列：状态（用于过滤"未交"学生）
- 文件名建议包含章节名称，如：`第一章 网络安全基础23-3.xlsx`

## 输出结果

### 1. 控制台输出
- 字数统计信息（总人数、字数分布区间）
- 相似度排名（学号、姓名、平均相似度、分数）
- 有图片的学生人数

### 2. 更新后的 Excel 文件
- 文件名：`<原文件名>_updated.xls`
- 格式：Excel 97-2003 (.xls)
- 内容：已删除"未交"状态行，已更新分数

### 3. 汇总统计文件
- 文件名：`summary_<章节名称>_<文件夹名>.txt`
- 内容：包含字数分布、图片统计等信息

## 评分算法

### 相似度计算
1. **文本相似度**：使用余弦相似度计算 TF-IDF 向量之间的相似度
2. **图片相似度**：
   - 两个文档都有图片：基于图片大小计算相似度
   - 只有一个文档有图片：相似度为 0%（表示完全不同）
   - 两个文档都没有图片：相似度为 100%（不影响综合相似度）
3. **综合相似度**：`综合相似度 = 文本相似度 × 0.8 + 图片相似度 × 0.2`

### 分数区间（正态分布）

| 相似度排名比例 | 分数 |
|--------------|------|
| 前 6.7%      | 100  |
| 6.7% - 15.9% | 95   |
| 15.9% - 30.9%| 90   |
| 30.9% - 69.1%| 85   |
| 69.1% - 84.1%| 80   |
| 84.1% - 93.3%| 75   |

*注：相似度越低，排名越靠前，分数越高*

## 技术细节

### 文本处理流程
1. 从 Word 文档中提取 XML 格式的文本内容
2. 使用 jieba 进行中文分词
3. 计算 TF-IDF 特征向量
4. 使用余弦相似度计算文本相似度

### 图片处理流程
1. 对于 `.docx` 文件：解压后从 `word/media/` 提取图片
2. 对于 `.doc` 文件：检查文件内容中的图片标签（`<w:binData>`、`<v:imagedata>`）
3. 提取图片大小作为特征
4. 计算图片大小相似度

### 异常处理
- 自动过滤 Word 临时文件（`~$` 开头）
- 自动跳过乱码和图片数据
- 自动处理"未交"状态的学生记录

## 注意事项

1. **文件编码**：确保 Word 文档使用 UTF-8 编码
2. **答案格式**：学生答案需要包含"学生答案："标记
3. **Excel 格式**：更新后的文件统一保存为 `.xls` 格式
4. **内存限制**：处理大量文档时可能需要增加内存
